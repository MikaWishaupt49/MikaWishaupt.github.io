Working on a function PWA

For the class of MDEV we have to create a PWA (progressive Web App)
I learned to create a web based app that is easy to user and to install without the usage of an app store!


The concept is called "Fair Estate" and is about finding afforable housing in the modern market. The PWA comes with a camera function (idea to scan codes from houses), map function (idea to find location of houses), share function (idea to share findings with others) and data storage to save the app on your device!
I learned to create something functonal and was the hardest course at CMD for development.
